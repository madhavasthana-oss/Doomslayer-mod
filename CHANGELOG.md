# Change Log

This file tracks all modifications made to the Bad Blood theme in pursuit of a more Doom Slayer–inspired desktop environment.

## Table of Contents

| Change    | Description                                                                                                                                                        |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Wallpaper | Added a custom wallpaper to the existing wallpaper collection depicting the Doom Slayer standing triumphant atop the corpses of demons in Hell.                    |
| Fastfetch | Modified Fastfetch to adopt a red command-terminal aesthetic and display the Mark of the Doom Slayer as its primary logo.                                          |
| Shell     | Customized the Oh My Posh **Tokyo** theme to better align with the Fastfetch color palette while retaining its information-dense and minimalist design philosophy. |

## Notice

Before creating new scripts, check the [`Scripts`](./Scripts) directory for existing utilities that can be extended or repurposed. This helps maintain consistency and reduces unnecessary duplication.
