# Bad Blood — Doom Slayer Customization Log

This repository contains modifications to the HyDE **Bad Blood** theme to create a darker, more industrial, Doom Slayer–inspired aesthetic.

## Files

### [`CHANGELOG.md`](./CHANGELOG.md)

Tracks every modification made to the theme.

Contents include:

* Configuration changes
* UI redesign decisions
* Asset replacements
* Theme experiments
* Bug fixes
* Future plans and roadmap items

---

### [`toolinfo.sh`](./scripts/toolinfo.sh)

Utility inspection and information broker script.

Used to:

* Gather information about installed utilities
* Query package metadata
* Inspect libraries and dependencies
* Assist with debugging and theme integration

---

### [`TOKENS.txt`](./TOKENS.txt)

Tracks token usage, references, and miscellaneous project metadata.

---

## Design Direction

Current objectives:

* Fortress of Doom aesthetic
* Industrial military styling
* Minimalist command-terminal appearance
* Red, black, and orange accents
* Functional over decorative UI
* Consistent visual language across:

  * Hyprlock
  * Kitty
  * Fastfetch
  * Waybar
  * Wlogout
  * Rofi
  * AGS components
